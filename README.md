# Abdul Rehman | Technical SEO Specialist Portfolio

A premium, high-performance portfolio website for a Technical SEO Specialist, featuring integrated blog management, project case studies, and a custom Admin Dashboard.

## 🚀 Features
- **Modern UI/UX**: Built with Tailwind CSS and advanced animations.
- **Dynamic Content**: Blogs and Projects are served from a custom Node.js/Express backend.
- **Admin Dashboard**: Secure portal for managing inquiries, blog posts, and projects.
- **Automated SEO**: Optimized meta tags, Open Graph support, and schema markup.
- **Client Inquiries**: Custom lead capture system with project detailing (budget, store URL, etc.).

## 🛠 Tech Stack
- **Frontend**: HTML5, Tailwind CSS, JavaScript (Vanilla), Lucide Icons, Quill.js.
- **Backend**: Node.js, Express, Mongoose, Multer (Image uploads).
- **Database**: MongoDB.

## 📦 Getting Started

### 1. Prerequisite
Ensure you have Node.js and MongoDB installed on your system.

### 2. Setup
Navigate to the `advanced-backend` directory and install dependencies:
```bash
cd advanced-backend
npm install
```

### 3. Environment Variables
Create a `.env` file in the `advanced-backend` directory:
```env
PORT=5000
MONGO_URI=your_mongdb_uri
JWT_SECRET=your_secret_key
NODE_ENV=development
```

### 4. Run the Application
Start the development server:
```bash
npm run dev
```

The website will be served at `http://localhost:5000`.

## 📂 Project Structure
- `/advanced-backend`: Modern Node.js/Express server logic and API.
- `/admin`: Frontend source for the Admin Panel.
- `/index.html`: Main portfolio website.
- `/images`: Static image assets for the project.
